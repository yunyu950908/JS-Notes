## 说明
命令行查询天气情况

## 使用方法
- 安装
```
npm install -g weather-liyu
```
- 查询
```
//直接查询当前IP所在城市的天气
weather
```
![](https://ws1.sinaimg.cn/large/889b2f7fgy1fikyhx1oy8j207m03fwee.jpg)

```
//查询指定城市的天气
weather [current city]
```
